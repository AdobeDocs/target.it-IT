# Cercare e filtrare i contenuti

Cerca le risorse per parole chiave nella libreria [!UICONTROL Offerte] in [!DNL Adobe Target].

## Cerca offerte di codice per parola chiave

1. Fai clic su **[!UICONTROL Offerte]** > **[!UICONTROL Offerte codice]**.
1. Digitare le parole chiave desiderate nella casella **[!UICONTROL Cerca in]**.

   È possibile utilizzare il menu a discesa per filtrare in base a [!UICONTROL Nome offerta], [!UICONTROL Percorsi AEM] o [!UICONTROL Tag AEM].

   Puoi anche filtrare utilizzando le opzioni [!UICONTROL Tipo], [!UICONTROL Source] e [!UICONTROL Tipo AEM] facendo clic sull&#39;icona [!UICONTROL Mostra filtri] ![Mostra icona filtri](/help/main/assets/icons/Filter.svg) per perfezionare ulteriormente la ricerca.

## Cerca offerte di immagini per parola chiave {#section_2465A71BC95942588F586B1EC8B9E5DB}

1. Fai clic su **[!UICONTROL Offerte]** > **[!UICONTROL Offerte immagine]**.

1. (Facoltativo) Passa da [!UICONTROL Vista a schede] a [!UICONTROL Vista a elenco] e fai clic sull&#39;icona [!UICONTROL Vista a schede] (![Icona Vista a schede](/help/main/assets/icons/ViewCard.svg) ) o sull&#39;icona [!UICONTROL Vista a elenco] ( ![Icona Vista a elenco](/help/main/assets/icons/ViewList.svg) ) nell&#39;angolo superiore destro della libreria [!UICONTROL Risorsa].
1. Fai clic sull&#39;icona **[!UICONTROL Solo contenuto]** ( ![Icona Solo contenuto](/help/main/assets/icons/RailLeft.svg) ) nell&#39;angolo in alto a sinistra per visualizzare la casella di ricerca.
1. Nella casella di ricerca, digita una parola chiave per le risorse da individuare, quindi premi [!UICONTROL Invio].
1. (Condizionale) Utilizza il riquadro **[!UICONTROL Filtri e opzioni]** a sinistra per perfezionare la ricerca.

   Puoi filtrare per:

   * **[!UICONTROL Tipo file]**:
      * [!UICONTROL Immagini]
      * [!UICONTROL Documenti]
      * [!UICONTROL File multimediali]
      * [!UICONTROL Archivi]
   * **[!UICONTROL Dimensione file]**: utilizzare il dispositivo di scorrimento per selezionare le dimensioni di file desiderate: [!UICONTROL Più piccolo], [!UICONTROL Piccolo], [!UICONTROL Medium], [!UICONTROL Grande] o [!UICONTROL Più grande].
   * **[!UICONTROL Ultima modifica]**: utilizza il cursore per selezionare il periodo di tempo: [!UICONTROL Recente], [!UICONTROL Ora], [!UICONTROL Giorno], [!UICONTROL Settimana], [!UICONTROL Mese], [!UICONTROL Anno] o [!UICONTROL Tutto Assets].
   * **[!UICONTROL Stato approvazione]**: [!UICONTROL Approvato] o [!UICONTROL Rifiutato]
   * **[!UICONTROL Stato scadenza]**: [!UICONTROL Scaduto].
   * **[!UICONTROL Valutazione media]**: da 1 a 5.
   * **[!UICONTROL Orientamento]**: [!UICONTROL Orizzontale], [!UICONTROL Verticale] o [!UICONTROL Quadrato].
   * **[!UICONTROL Stile]**: [!UICONTROL Colore] o [!UICONTROL Bianco e nero].
   * **[!UICONTROL Punteggio utilizzo]**: [!UICONTROL Minimo] a [!UICONTROL Massimo].
   * **[!UICONTROL Punteggio impression]**: [!UICONTROL Minimo] a [!UICONTROL Massimo].
   * **[!UICONTROL Punteggio clic]**: [!UICONTROL Minimo] a [!UICONTROL Massimo].
